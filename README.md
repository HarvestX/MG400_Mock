# MG400_Mock

Dobot MG400 Mock Server Package.
It can mimic DobotMG400 actions as mock server.

![Image](media/mg400_mock.gif)

## Requirements
- [Docker](https://docs.docker.com/get-docker/)
- [Docker Compose](https://docs.docker.com/compose/install/)



## Install

```console
cd MG400_Mock
make
```

## shutdown

```console
make down
```
